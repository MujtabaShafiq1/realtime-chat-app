# CHAT APP

## Overview

Allows user to communicate in private or group chat in real time.

## Features

- Login / Signup
- Email Verification
- Search User
- Private Chat
- Group Chat
- User online/offline status
- User typing status
- Latest Message
- Message Reading
- Message Deletion
- Send Images
- Image Slider
- Group Admin can Add User
- Group Admin can Remove User
- Leave Group Chat
- Informative Message in Group Chat
- Message Interval Information
- Dark Theme / Light Theme

## Technologies

- React
- Express
- Node
- Mongodb + Mongoose
- Socket.io

## ENV

- ### Client

- REACT_APP_SERVER

- ### Server

- PORT
- JWT_KEY
- JWT_EMAIL_KEY
- MONGO_URI
- CLIENT
- SERVICE = gmail
- EMAL_PORT
- SECURE = true
- USER
- PASS

- ### Socket

- PORT
- JWT_KEY
- CLIENT
