# CHAT APP

## Overview

Allows user to communicate in private or group chat in real time.

## Features

- Login / Signup
- Email Verification
- Search User
- Private Chat
- Group Chat
- User online/offline status
- User typing status
- Latest Message
- Message Reading
- Message Deletion
- Send Images
- Image Slider
- Group Admin can Add User
- Group Admin can Remove User
- Leave Group Chat
- Informative Message in Group Chat
- Message Interval Information
- Dark Theme / Light Theme

## Technologies

- React
- Express
- Node
- Mongodb + Mongoose
- Socket.io
